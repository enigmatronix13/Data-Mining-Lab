# Data Mining Lab 

This repository contains implementations, experiments, and query solutions developed as part of the Data Mining Lab course. It covers practical work involving ETL processes, data preprocessing, data mining algorithms, and SQL-based analytical queries.


## Data Integration (ETL)

- Implementation of ETL workflows using Talend Open Studio
- Data extraction from structured files
- Data transformation (filtering, mapping, aggregation)
- Data loading into output formats

Key components used:
- tFileInputDelimited
- tMap
- tFilterRow
- tAggregateRow
- tSortRow


## Data Mining Algorithms

- Apriori Algorithm (Association Rule Mining)
- K-Means Clustering
- Decision Tree (ID3)
- Naïve Bayes Classifier


## SQL and Database Queries

Includes solutions to structured query problems based on relational schemas such as:

- Student, Course, Enroll
- Book Adoption and Textbook datasets

Representative queries include:
- Filtering datasets based on conditions
- Aggregation and grouping operations
- Join-based queries across multiple tables
- Identifying patterns such as maximum enrollments and publisher usage


## Key Concepts

- Data preprocessing and cleaning
- ETL pipeline design
- Classification and clustering techniques
- Association rule mining
- Relational database querying


## Technologies Used

- Talend Open Studio
- RapidMiner
- SQL (MySQL/PostgreSQL)
- CSV and structured datasets
