# Data Mining Lab

This repository contains practical implementations and experimental results for the Data Mining Lab course, focusing on ETL processes, algorithmic analysis, and relational data querying.

## Core Modules

* **ETL Workflows:** Data integration using Talend Open Studio, including extraction, transformation (mapping, filtering, aggregation), and loading.
* **Data Mining Algorithms:** Implementation of Apriori (Association Rules), K-Means (Clustering), ID3 Decision Trees, and Naïve Bayes Classifiers.
* **Analytical SQL:** Query solutions for relational schemas, involving complex joins, data aggregation, and pattern recognition.

## Tools and Technologies

* **ETL:** Talend Open Studio
* **Analytics:** RapidMiner
* **Database:** SQL (MySQL/PostgreSQL)
* **Format:** CSV and structured datasets
