# Lab 6: Apriori Algorithm

## Objectives

* To implement the Apriori algorithm for mining frequent itemsets
* To generate association rules using minimum support and confidence
* To understand and implement partition-based Apriori for file-based datasets


## Lab Exercise

1. Find the frequent itemsets using the Apriori algorithm for a given transactional database.
2. Generate association rules by considering suitable minimum support and confidence values.


## Additional Exercise

1. Implement partition-based Apriori algorithm for a database stored as a file and find the association rules.
